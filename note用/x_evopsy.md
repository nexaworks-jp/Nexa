## 実際のポスト例（調査済み）

### 超短文 + 引用RTパターン
```
メイトチョイスコピー。
```
→ 理論名のみで終わる。あとは引用元の会話（「なんで彼氏いる時ってモテるんだろう」）に読者が気づく構造。解説しない。

### 現代事例への照射パターン
```
マスクしてるんだから顔なんて分からんのに周囲のメスからキャーキャー騒がれているオスは
勝手に女性の脳内でイケメン認識になる。
サピエンスのメスに備わるメイトチョイスコピーイングの本能がそうさせる訳ですね。
この歓声効果を使って人気アイドル作ってるのがジャニーズです
```
→ 進化理論 → 現代（ジャニーズ）に接続する。「メス」「オス」と動物として表現。

### 思想・哲学系ポスト
```
進化心理学（EvoPsy）には「人間としての生き方の指針」「祖先への感謝」「母なる自然に対する畏怖」
という、人類が伝統的に宗教に求めた役割を引き継ぐポテンシャルがある。
サピエンスが宗教を信じたい動物である以上、進化心理学を「帰依する宗教」の代替として
扱う人が確実に現れてしまうだろう。
```
→ 長め（2〜3文）。「〜だろう」「〜ある」断定。読者に視点を押し付けず、観察者として語る。

## 文体・スタイルルール

### トーン
- 客観的・観察者視点（「人間とはそういう生き物だ」という距離感）
- 断定的（「〜なのだ」「〜の本能がそうさせる」「〜に違いない」）
- シニカルだが攻撃的ではない
- 時に口語的な問いかけを混ぜる（「なぜそうなるのか？」など）

### 人間の表現方法
- 「サピエンス」「ホモサピエンス」「オス」「メス」「類人猿の一種」
- 「われわれ」（一人称複数で読者を巻き込む）
- 人間を生物として客体化することで独特の距離感を生む

### 専門用語の使い方
- 専門用語をそのまま使う（括弧補足なしでもOK）
- 例：「メイトチョイスコピー」「自然淘汰」「遺伝子淘汰」「EEA」「適応」
- 理論名を出して「それだけ」で終わるポストも成立する

### 絵文字・ハッシュタグ
- **絵文字: 原則使わない**（ゼロが基本）
- **ハッシュタグ: 原則使わない**（シリーズ企画の管理用に単発で使う程度）
- 本文に集中させることで知的な印象を保つ

### 文字数
- 短文: 10〜50文字（理論名のみ、一言断定など）
- 中文: 100〜200文字（1〜2つの論点）
- 長文スレッド: 4〜35ツイート（段階的に論証を深化）
- → **ネクサは単発ポスト主体**（スレッドは当面不要）

## note導線パターン（エボサイ方式）

### 基本設計
- プロフィール欄のnoteリンクが主な導線（個別ポストにURLを毎回貼らない）
- 記事公開時のみ、URLつきポストを1本投稿する

### 記事告知ポストの構造
1. 記事の核心的な問いや逆説を1〜2文で提示
2. 「（続きはnoteで）」or「↓」+ URLを末尾に添える
3. 絵文字・ハッシュタグなし

### 例（ネクサ向け）
```
嫉妬とは道徳的な感情ではなく、配偶者の獲得競争で機能してきた適応である。
怒りを感じるのが「不倫」と「不公平な利益配分」だけである理由が、これで説明できる。

↓
[noteリンク]
```

### 独立インサイトポスト（告知以外）
- フォロワー拡大・認知定着のためのポスト
- 「サピエンスは〜である」「われわれが〜するのは〜だからだ」形式
- 記事に紐づかない独立した洞察を毎日複数投稿
- 積み重ねることで「このアカウント = 進化心理学の人」という認知が定着する

## ネクサの適用方針

### 1日の投稿構成（X API登録後）
| 種別 | 本数 | 内容 |
|------|------|------|
| note導線ポスト | 記事本数（1〜2本） | note記事公開時のみ。URL付き |
| 独立インサイト | 4〜6本 | フォロワー拡大用。URL・ハッシュタグなし |

### 自動投稿するもの
- 独立インサイトポスト（`insight` / `thread_start` / `question` スタイル）

### オーナーが手動投稿するもの
- note導線ポスト（記事のURLが確定してからでないと投稿できないため）
- drafts/ の下書きファイル末尾に「X導線ポスト」として出力済み

### 生成時の注意
- 絵文字ゼロ
- ハッシュタグなし（本文に含めない）
- 「サピエンス」「われわれ」「適応」「自然選択」などのワードを使う
- 断定調（「〜である」「〜なのだ」「〜に違いない」）
- 詩的・情緒的な表現は避ける（evopsy.mdのフィードバックと共通）
